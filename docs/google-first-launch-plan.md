# GGI Solutions Google-First Website Launch Plan

## Executive Recommendation

GGI Solutions should launch with the current Next.js website as the public production site, while using Google products around it for business operations.

This gives the company a professional website while still keeping the workflow familiar and Google-centered.

## Simple Decision

```text
Website: Next.js + Vercel
Business operations: Google Workspace
AI prototyping: Google AI Studio / Gemini
Lead capture: Google Forms first, then upgrade later
Analytics: Google Analytics / Search Console
```

## Why This Is Best

Google products are excellent for business operations, forms, files, email, AI testing, and analytics.

But the public GGI Solutions website should live on a real web hosting/deployment platform so it can look premium, scale, and support custom features like smart qualifiers.

## What Not To Do

Do not rebuild the whole website inside Google AI Studio.

Google AI Studio is the lab for the AI brain. It is not the best place to host the company website.

## Launch Phases

## Phase 1: Publish the Website

Use the existing app.

Pages already started:

- homepage
- services
- solution finder
- custom qualifiers
- contact
- event page template
- follow-up system page

Action:

```text
Deploy to Vercel.
```

## Phase 2: Connect Google Intake

Create a Google Form called:

```text
GGI Solutions Project Inquiry
```

Fields:

```text
Name
Email
Phone
Organization
Organization Type
What do you need help with?
Budget range
Timeline
Tell us what you are trying to build
```

Connect the form to a Google Sheet.

## Phase 3: Add Form Link to Website

Until the site has a true backend form, link buttons to the Google Form.

This allows the company to start collecting leads immediately.

## Phase 4: Add Google Analytics

Create Google Analytics property.

Connect it to the website.

Add Search Console after the domain is connected.

## Phase 5: Upgrade Later

After leads start coming in, upgrade to:

```text
Supabase + Resend + dashboard
```

This can happen after launch, not before.

## Domain Recommendation

Options:

```text

```

If the exact domain is not available, use:

```text

```

## CEO Rule

Do not delay launch because the perfect backend is not ready.

Use Google Forms first.

Launch the website.

Collect leads.

Upgrade the tech after the first clients come in.

## Final Stack

```text
Public Site: Next.js on Vercel
Domain: Google Domains/Squarespace Domains or another registrar
Email: Google Workspace
Forms: Google Forms
Lead Tracking: Google Sheets
Files: Google Drive
AI Brain: Google AI Studio / Gemini
Analytics: Google Analytics + Search Console
```

## Final Recommendation

Launch GGI Solutions using the current site and a Google-first operations layer.

This keeps the launch professional, scalable, and familiar.
