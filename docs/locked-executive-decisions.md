# Locked Executive Decisions for GGI Solutions

## Purpose

This document is the current source of truth for GGI Solutions decisions, platform direction, product positioning, and launch strategy.

These decisions should guide future builds, recommendations, and strategy unless Reno explicitly changes direction.

---

```text
GGI Solutions
```

GGI Solutions produces digital systems, websites, smart qualifiers, messaging worlds, follow-up systems, and growth development tools.

## Launch Stack Decision

Use this stack as the default recommendation:

```text
GitHub = code home
Vercel = production website deployment
Hostinger = domain / DNS
Google = business operations
Google AI Studio = AI qualifier logic and Gemini prototyping
Replit = experiments, bots, prototypes, and quick scripts
```

---

## Production Website Decision

The public GGI Solutions website should be deployed from the current Next.js app in GitHub to Vercel.

Do not rebuild the main website in:

```text
Google AI Studio
Replit
Hostinger Builder
Google Sites
```

unless there is a clear business reason.

Reason:

```text
The current site is already built in Next.js and Vercel is the cleanest professional launch path for a Next.js website.
```

---

## Google Product Role

Google products remain central to the business workflow.

Use Google for:

- Google Workspace email
- Google Forms for initial project inquiries
- Google Sheets for lead tracking
- Google Drive for client assets
- Google Calendar for bookings
- Google Analytics and Search Console for tracking
- Google AI Studio / Gemini for qualifier AI logic

Rule:

```text
Google runs the business operations. Vercel hosts the website.
```

---

## Hostinger Role

Hostinger should be used for:

- domain registration
- DNS management
- possible email if Google Workspace is not used

Hostinger should not be the primary host for the current Next.js site unless needed.

---

## Replit Role

Replit should be used for:

- learning
- experiments
- quick scripts
- small bots
- temporary APIs
- demos

Replit should not be the primary host for the GGI Solutions company website.

---

## Vercel Decision

Vercel is the recommended production deployment platform for the current site.

Reason:

- built for Next.js
- connects to GitHub
- automatic deployments
- preview links
- supports future forms, APIs, and AI features
- professional launch path

Executive decision:

```text
Launch the GGI Solutions website on Vercel.
```
