# Track 1: Church Follow-Up System

## Core Offer

A simple AI-assisted follow-up system that helps churches track visitors, follow-up actions, care needs, and next steps without replacing human pastoral care.

## Target Customer

Small to mid-sized churches that struggle with:

- first-time visitor follow-up
- inconsistent communication
- no clear next-step process
- no visibility for pastors or ministry leaders
- people falling through the cracks

## Simple Positioning Statement

GGI Hub helps churches stop losing people after the first visit by giving them a simple follow-up system that tracks visitors, reminders, care needs, and next steps in one place.

## What the Client Gets

### Basic Version

- Visitor intake form
- Follow-up tracker
- Leader dashboard
- Next-step status labels
- Weekly follow-up report
- Simple email/SMS workflow recommendations

### Premium Version

- Everything in Basic
- AI-assisted follow-up suggestions
- Care flags
- New member pathway
- Volunteer/leader assignment tracking
- Monthly ministry health report

## Suggested Pricing

### Starter Setup

```text
$250-$500 setup
$75-$150/month maintenance
```

Best for very small churches.

### Growth Setup

```text
$750-$1,500 setup
$200-$350/month maintenance
```

Best for churches that need dashboard + follow-up automation.

### Custom Ministry System

```text
$2,500+ setup
$500+/month support
```

Best for churches that want follow-up, onboarding, leadership pipeline, and care tracking.

# Combined Strategy

These two tracks should work together.

## Church Follow-Up System Opens the Door

A church may start with an event page for:

- revival
- worship night
- conference
- youth event
- anniversary
- outreach

Then after the event, GGI Hub can offer the Church Follow-Up System to help them track visitors, attendees, and next steps.

## Event Pages Create Fast Cash

Event pages are easier to sell because they are simple and urgent. Someone has an event coming up and needs a link, a form, and clean promotion.

## Follow-Up System Creates Recurring Revenue

The follow-up system is stronger long-term because it can create monthly maintenance revenue.
